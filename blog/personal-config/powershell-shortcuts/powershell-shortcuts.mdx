## PowerShell profile

```powershell
# Variables

$localRepo = $HOME + '\source\repos'

# Functions

function goToRepos { cd $localRepo }

# Alias

New-Alias -Name startTaskManagerAPI -Value $localRepo\TaskManager\startTaskManagerScript.ps1
```
